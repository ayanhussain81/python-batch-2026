## Overview

**3 months · 24 sessions** in numbered folders.

**For:** beginners and anyone who wants a clear path to a small portfolio project.  
**You need:** a computer you can install software on; no prior coding required.

**How we run it:** concept → exercises → homework from the session folder. AI is for speed and explanation, not a substitute for understanding what you submit.

**Demo day (session 24):** short walkthrough of your final project—what it does, how you built it, one improvement you’d make next.

## Modules

```
Month 1     →  M2 wks 1-2  →  M2 wks 3-4  →  Month 3
──────────────────────────────────────────────────────
Module 1    Module 2       Module 3       Module 4
Core Python Practical      OOP & real use AI-assisted coding
Sessions 1-8  9-14         15-18          19-24
```

| Module | Focus |
|--------|--------|
| **1** | Syntax, types, control flow, functions, small programs |
| **2** | Files, modules, errors, scripts you can reuse |
| **3** | Classes, objects, modeling data; stdlib as needed |
| **4** | Cursor + Claude Code: build and refactor *with* help—you own the code |
